# todo-app-mael
**🔗 Demo Live:** https://mael07sss.github.io/todo-app-mael

App de tareas nivel producción hecha en 19 días con JavaScript puro.

### ✨ Features
- ✅ **Drag & Drop** - Reordena tareas arrastrando
- ✅ **Dark Mode** - Persistente con LocalStorage
- ✅ **Filtros** - Todas / Pendientes / Completadas
- ✅ **Contador** - Progreso en tiempo real
- ✅ **Edición inline** - Doble click para editar
- ✅ **Fechas** - Registro de creación automático
- ✅ **100% Responsive** - Mobile first

### 🛠️ Tech Stack
`HTML5` `CSS3` `JavaScript ES6+` `LocalStorage API` `Drag & Drop API`

### 🎯 Lo que aprendí
- Manipulación avanzada del DOM
- Event Delegation y Bubbling
- Persistencia de datos sin backend
- UI/UX con CSS puro
- Git, GitHub y Deploy con GitHub Pages
